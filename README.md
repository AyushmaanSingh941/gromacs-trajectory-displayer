# gromacs-trajectory-displayer

Designed to be simple and extensible, this project focuses on fast, clear visual inspection of molecular dynamics trajectories and basic exploratory workflows. It works with common GROMACS formats and can be used from the command line or embedded into other tools and scripts.

Features
- Simple, minimal UI focused on quick visual inspection and exploratory tasks
- Extensible codebase so additional importers, viewers, or analysis modules can be added
- Intended to work well with common GROMACS files (e.g., .xtc, .trr, .gro, .pdb)

Getting started
1. Clone the repository:

   git clone https://github.com/AyushmaanSingh941/gromacs-trajectory-displayer.git

2. See the source files and examples in the repository for usage details. If you don't see any code yet, this repo currently contains documentation and scaffolding — see the Issues tab to track planned features.
